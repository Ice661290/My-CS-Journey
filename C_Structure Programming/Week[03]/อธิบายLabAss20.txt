1. โครงสร้างของ for loop 
 ปกติแล้ว for จะมี 3 ส่วนคือ for (เริ่มต้น; เงื่อนไข; เปลี่ยนแปลงค่า)
      ส่วนเริ่มต้น (Init): ว่างไว้ ;
         -เพราะเรามีค่า n ที่รับมาจาก scanf อยู่แล้ว ไม่ต้องประกาศตัวแปรนับรอบ (i) 
      เพิ่มส่วนเงื่อนไข (Condition): n != 0
         -หมายความว่า "ให้ทำไปเรื่อยๆ ตราบใดที่ n ยังไม่กลายเป็น 0" (ยังมีตัวเลขเหลือให้ตัด)
      ส่วนเปลี่ยนแปลงค่า (Update): n /= 10
         -หมายความว่า "หลังจากจบรอบ ให้ตัดเลขตัวท้ายของ n ทิ้งไป" (หาร 10 แบบจำนวนเต็ม คือการตัดหลักหน่วยทิ้ง)

2. จำลองการทำงาน (Trace) ทีละรอบสมมติว่าเราใส่เลข 123 เข้าไป (n = 123, reversed = 0)
รอบที่ 1:เช็คเงื่อนไข: n (123) ไม่เท่ากับ 0 -> ทำในลูป
ในลูป:
  remainder = 123 % 10 ได้ 3 (ดึงเลขท้ายสุดมา)
  reversed = 0 * 10 + 3 ได้ 3 (เอาเลข 3 ไปแปะไว้)
  จบในลูป: ทำคำสั่งส่วนเปลี่ยนแปลงค่า -> n /= 10 (123 หาร 10 เหลือ 12)
รอบที่ 2:เช็คเงื่อนไข: n (12) ไม่เท่ากับ 0 -> ทำในลูป
ในลูป:
  remainder = 12 % 10 ได้ 2 (ดึงเลขท้ายสุดของรอบนี้มา)
  reversed = 3 * 10 + 2 ได้ 32 (เอา 3 เดิมขยับไปหลักสิบ แล้วเอา 2 มาต่อท้าย)
  จบในลูป: ทำคำสั่งส่วนเปลี่ยนแปลงค่า -> n /= 10 (12 หาร 10 เหลือ 1)
รอบที่ 3:เช็คเงื่อนไข: n (1) ไม่เท่ากับ 0 -> ทำในลูป
ในลูป:
  remainder = 1 % 10 ได้ 1
  reversed = 32 * 10 + 1 ได้ 321 (เอา 32 ขยับไป แล้วเอา 1 มาต่อท้าย)
  จบในลูป: ทำคำสั่งส่วนเปลี่ยนแปลงค่า -> n /= 10 (1 หาร 10 เหลือ 0)
รอบที่ 4:เช็คเงื่อนไข: n เป็น 0 แล้ว -> หยุดทำ (ออกจากลูป)
ผลลัพธ์: ตอนนี้ค่า reversed คือ 321 ซึ่งเป็นการกลับด้านของ 123 ครับ
